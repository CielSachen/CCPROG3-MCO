# Coffee Truck Simulator

## Compilation

```bash
javac ./**/*.java
```

## Usage

> [!IMPORTANT]
> The working directory is the `src/main/java` folder; therefore, you must run the following command inside of the `src/main/java` folder:
>
> ```bash
> cd ./src/main/java/
> ```

```bash
# For MCO1:
java cielsachen.mco1.Main
```

## Documentation

> [!IMPORTANT]
> The working directory is the `src/main/java` folder; therefore, you must run the following command inside of the `src/main/java` folder:
>
> ```bash
> cd ./src/main/java/
> ```

```bash
# For MCO1:
javadoc cielsachen.mco1 -d ../../../docs -subpackages cielsachen.mco1.controller cielsachen.mco1.exception cielsachen.mco1.helper cielsachen.mco1.model cielsachen.mco1.model.coffee cielsachen.mco1.service cielsachen.mco1.util
```
